## Requirement

- nginx 1.10 or above
- php 7.0 or above
- mysql 5.7 or above
- Laravel 5.3

## Installation
1. Install git and composer on your environment.
1. Copy git code from Github.
1. Run 'composer update' in your downloaded directory.
1. Congratulations! Check your site.
